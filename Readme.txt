Programa 1 ------------------------------

Se despliega un Menú básico, el que le dará las opciones de leer una nota, escribir o borrarla, además de salir del script

Si lee, deberá elegir la fecha de la nota que quiera leer
Si escribir, deberá ingresar la nota, y después la fecha de esta
Si elimina, deberá elegir la fecha de las que se desplieguen, para eliminar
Si salir, saldrá del bucle while y saldrá del programa.

Programa 2 ------------------------------
Se desarrollan dos funciones: una para calcular el factorial de n y otra para aproximar la función arcsin mediante series de Taylor. Ambas funciones trabajan en conjunto.
Inicialmente, se define un arreglo con los valores (4,8,12) para el primer gráfico.El grafico 1 también incluirá graficada la funcion arcsen mas exacta proporcionado por NumPy, lo cual permitirá comparar estos valores con las aproximaciones obtenidas mediante las series de Taylor para los diferentes valores de N.
La segunda parte, N se setean con otros valores que nos piden, y se crea una figura con axs para subplotear cada valor nuevo y compararlo con arcsen, con un subplot de tamaño (2x2) se van ploteando uno a uno, hasta que final mente se despliegan todos los gráficos como se puede visualizar
